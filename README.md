# SpringBoot RESTful API Library API

### Features:
- Multiple profiles
- Exception handling
- In memory implementation
- Database implementation (H2 and PostgreSQL)
- SpringBoot Actuator
- Tested using client API: Bruno

### How to run:
Run com.assignment.library_api.LibraryApiApplication java file
